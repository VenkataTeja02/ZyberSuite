API Setup

Configure your VirusTotal API key — used by Malware Scanner and URL Detector

Browser limitation: Your API key is stored in localStorage and sent directly from your browser to VirusTotal. It is readable via DevTools → Application → Local Storage and visible in the Network tab. Do not use a paid or high-quota key in shared or public environments. For production, route requests through a backend proxy that holds the key server-side.
